# SAHO

## Sistema de Administración Médica Hospitalaria

SAHO es una plataforma web que utiliza Inteligencia Artificial para facilitar la gestión y comprensión de la información médica.

### Funcionalidades

- Asistente médico con IA
- Interpretación de recetas (OCR)
- Análisis de exámenes clínicos
- Historial clínico digital
- Grabación y transcripción de consultas
- Recomendación de especialistas
- Árbol genealógico de salud
- Plataforma SaaS

## Sitio Web

https://sites.google.com/view/saho-explica/inicio

## Tecnologías

- Google Sites
- Inteligencia Artificial
- OCR
- Cloud Computing
- SaaS

## Objetivo

Brindar una plataforma moderna que ayude a pacientes y profesionales a gestionar información médica de manera segura y eficiente.
